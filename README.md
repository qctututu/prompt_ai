# 🦞 Prompt AI — Prompt Engineering 学习与实践

CAIE AI Level I 备考 · Prompt 模板库

## 📂 目录结构

```
prompt_ai/
├── README.md               # 项目说明
├── .gitignore              # 忽略文件
├── templates/              # Prompt 模板
│   ├── basic/              #   ├── 基础篇：零样本 / 少样本 / 指令
│   └── advanced/           #   └── 进阶篇：角色扮演 / 链式思维 等
├── playground/             # Playground 测试记录与输出样本
├── scripts/                # API 调用脚本
└── notes/                  # 学习笔记
```

## 📝 使用方式

1. 打开对应模板文件，复制 Prompt 模板
2. 将 `{{变量}}` 替换为你的实际内容
3. 在 Playground 或 API 中测试
