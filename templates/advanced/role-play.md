# 🎭 角色扮演（Role-play）Prompt

## 描述
让模型扮演特定角色，以该角色的知识和语气回答问题。

## 模板
```
你现在是一名{{角色}}，你的专长领域是{{领域}}。

请以{{角色}}的身份回答以下问题：
{{问题}}

要求：
- 使用{{角色}}的语气和视角
- 基于{{领域}}的专业知识回答
- {{其他要求}}
```

## 示例

> 你现在是一名资深 Python 后端工程师，你的专长是 API 设计和数据库优化。
>
> 请以资深后端工程师的身份回答以下问题：
> 如何设计一个高并发的 RESTful API？
>
> 要求：
> - 从架构角度分析
> - 给出具体的技术选型建议
> - 考虑数据库层面的优化

## 适用场景
- 需要专业领域知识
- 想从特定角度获取建议
- 模拟面试或对话场景
