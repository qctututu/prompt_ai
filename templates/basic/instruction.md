# 🔵 指令式（Instruction）Prompt

## 描述
明确告诉模型角色、任务、输出格式要求，精确控制输出。

## 模板
```
你是一名{{角色}}。

任务：{{任务描述}}

要求：
- {{要求1}}
- {{要求2}}
- {{要求3}}

输出格式：
{{输出格式说明}}
```

## 示例

> 你是一名职场助理，请帮我写一封正式的请假邮件。
> 要求：
> - 收件人：王经理
> - 请假人：小刘
> - 请假时长：1天
> - 请假原因：去医院体检
> - 语气：礼貌、正式
> - 输出格式：主题 + 正文

## 适用场景
- 复杂、多约束任务
- 需要精确控制输出格式
- 对角色和专业性有要求
