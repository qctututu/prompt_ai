# 📝 Prompt Engineering 学习笔记

## 基本原则
- 清晰 > 模糊
- 具体 > 抽象
- 指令 > 暗示
- 示例 > 描述

## 常用技巧
| 技巧 | 说明 | 适用 |
|------|------|------|
| 零样本 | 直接提问 | 简单任务 |
| 少样本 | 给示例 | 格式控制 |
| 指令式 | 结构化指令 | 复杂任务 |
| 角色扮演 | 设定角色 | 专业领域 |
| 链式思维 | 逐步推理 | 逻辑/数学 |
